Berkeley DB: DbLsn

DbLsn

#include <db_cxx.h>

Description

A DbLsn is a log sequence number that is fully encapsulated. The class itself has no methods, other than a default constructor, so there is no way for the user to manipulate its data directly. It is initialized and used by methods in DbLog and DbMpool. These methods always accept a pointer to a DbLsn.
