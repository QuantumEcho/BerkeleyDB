Berkeley DB Reference Guide:
Introduction

What does the Berkeley DB distribution include?

The Berkeley DB distribution includes complete source code for the Berkeley DB library, including all three Berkeley DB products and their supporting utilities, as well as complete documentation in HTML format.

The distribution does not include prebuilt binaries or libraries, or hard-copy documentation. Prebuilt libraries and binaries for some architecture/compiler combinations are available as part of Sleepycat Software's Berkeley DB support services.
