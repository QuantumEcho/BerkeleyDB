Berkeley DB: Access method operations

Access method operations

Once a database has been opened using db_open, and a handle on that database returned, there are seven standard access method operations. Each of these operations is performed using a function that is referenced from the returned handle. The seven operations are as follows:

DB->get
Retrieve a record
DB->put
Store a record
DB->del
Delete a record
DB->cursor
Open a cursor into the database
DB->stat
Return statistics about the database
DB->sync
Flush the underlying cache
DB->close
Close the database
