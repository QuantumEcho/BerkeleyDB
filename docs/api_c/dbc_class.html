DBC

 

#include <db.h>


typedef struct __dbc DBC;

Description: DBC

The DBC object is the handle for a cursor into a Berkeley DB database. The handle is not free-threaded. Cursor handles may be used by multiple threads, but only serially, that is, the application must serialize access to the DBC handle.

If the cursor is to be used to perform operations on behalf of a transaction, the cursor must be opened and closed within the context of that single transaction. Once DBcursor->c_close has been called, the handle may not be accessed again, regardless of the method's return.
