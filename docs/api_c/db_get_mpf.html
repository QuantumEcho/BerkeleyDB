DB->mpf

 

#include <db.h>


DB_MPOOLFILE *DB->mpf

Description: DB->mpf

The DB->mpf method returns the Db.get_mpf.

The DB->mpf method may be called at any time during the life of the application.

DB->mpf gives access to the DB_MPOOLFILE associated with a DB object.
