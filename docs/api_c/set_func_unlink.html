DBENV->set_func_unlink

 

#include <db.h>


int
DBENV->set_func_unlink(DB_ENV *dbenv,
    int (*func_unlink)(const char *path));

Description

Replace Berkeley DB calls to the IEEE/ANSI Std 1003.1 (POSIX) unlink function with func_unlink, which must conform to the standard interface.

The DBENV->set_func_unlink interface affects the entire application, not only the environment in which it is called.

The DBENV->set_func_unlink interface may only be used to configure Berkeley DB before the DBENV->open interface is called.

The DBENV->set_func_unlink function returns a non-zero error value on failure and 0 on success.

Errors

EINVAL
An invalid flag value or parameter was specified.

Called after DBENV->open was called.
