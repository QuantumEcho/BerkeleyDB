Berkeley DB: DbEnv.get_version_major

DbEnv.get_version_major

import com.sleepycat.db.*;


public static int get_version_major();
public static int get_version_minor();
public static int get_version_patch();
public static String get_version_string();

Description

These methods return version information about the underlying DB software. The lower level DB software is released with a major, minor and patch number, which is returned by get_version_major, get_version_minor and get_version_patch. A verbose version of this version information, suitable for printing is returned by get_version_string.
